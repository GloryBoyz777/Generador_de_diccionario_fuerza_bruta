# Generador de diccionarios by Gloryboyz77

## Descripción
Este script funciona para generar posibles passwords, teniendo distintos datos de el user, por ejemplo su año de nacimiento, nickname, color favorito, etc. Se puede seguir modificando con el fin de que sea mas personalizado, en un futuro se agregaran mas utilidades

## Características 
* Genera de forma dinamica las passwords
* Evita crear passwords duplicadas
* Exporta el archivo de forma personalizada a conveniencia
* Generador ligero y facil de usar

## Uso
1. Clona este repositorio.
2. Ejecuta el script con `python generador.py` (o el nombre que le hayas puesto).
3. Responde a las preguntas interactivas (puedes presionar Enter para omitir las que no sepas).
4. Introduce el nombre deseado para tu archivo de salida.
